Core Library
Vivarium Core

This is the heart of the Vivarium Collective. Its modular interface allows for easy integration of systems biology models, and its simulation engine can execute the resulting composite model.

Official Project
Vivarium Chemotaxis

Vivarium Chemotaxis is a library for the multi-scale model of chemotaxis described in Agmon, E.; Spangler, R.K. A Multi-Scale Approach to Modeling E. coli Chemotaxis. Entropy 2020, 22, 1101 .
